# Research Gap Closure Plan

The attached report establishes the HNB concept, five-agent pipeline, and execution-agent framing. This repository upgrade closes the main rigor gaps as follows:

- Benchmark realism: move from UCI-only proxies to Amazon Reviews, RetailRocket, and Instacart.
- Systems realism: move from local JSON state to PostgreSQL, Redis, workers, and metrics.
- Reproducibility: versioned configs, Dockerized stack, experiment tracking, and tests.
- Explainability: preserve inspectable expert contributions while expanding to richer context features.
- Failure analysis: formalize execution failure categories and telemetry instead of anecdotal logging.
